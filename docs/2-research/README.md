# 2-research

Processed research with context added. Source material that informs decisions.

| Subfolder | What goes here |
|---|---|
| `competitors/` | Competitor analysis: pricing, features, UX patterns. One file per competitor. |
| `references/` | External articles, docs, and links worth keeping. Include a summary — links rot. |
| `tech-options/` | Technology comparisons before a decision is made. Framework, library, or service evaluations. |

**Rule:** Each file should answer: *what did we find, and what does it mean for this project?*

AI agents use this folder to understand the landscape before proposing architecture or feature decisions.
