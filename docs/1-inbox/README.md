# 1-inbox

Raw capture — dump everything here first, process it later.

| Subfolder | What goes here |
|---|---|
| `ideas/` | Quick notes, shower thoughts, feature sparks. One idea per file, rough prose is fine. |
| `screenshots/` | UI references, competitor captures, error screenshots. Name by date + subject: `2024-01-15-stripe-checkout.png` |
| `voice-notes/` | Transcripts from voice recordings. Raw text is fine — accuracy matters, polish doesn't. |

**Rule:** Nothing gets edited here. Process means moving to `2-research/`, `3-design/`, or `4-builds/` with context added.

AI agents can read this folder for raw context and unprocessed requirements.
