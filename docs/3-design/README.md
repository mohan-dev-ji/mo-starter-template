# 3-design

Design artifacts. Visual and UX intent before it becomes code.

| Subfolder | What goes here |
|---|---|
| `sketches/` | Hand-drawn or rough digital sketches. Photos of whiteboard diagrams are fine. |
| `ui-notes/` | Written UI intent: spacing, hierarchy, interaction notes. Good for briefing AI agents on UX goals. |
| `wireframes/` | Structured wireframes — Figma exports, Excalidraw files, annotated screenshots. |

**Rule:** Include intent, not just the artifact. A wireframe without notes on what it's solving is just noise.

AI agents read this folder to understand design goals before generating or modifying components.
